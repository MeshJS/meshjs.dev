Server-side minting lets you create Cardano NFTs without a browserâ€”ideal for automated token distribution, airdrops, and backend JavaScript Web3 services. This guide shows you how to build a Node.js application using the Mesh Cardano SDK that mints multiple NFTs and sends them to different recipients in a single transaction.

## What Do You Need?

**Prerequisites:**
- <Link href="https://code.visualstudio.com/">Visual Studio Code</Link> or any code editor
- <Link href="https://nodejs.org/en">Node.js</Link> LTS version

## How Do You Set Up the Project?

Create a new folder and initialize a Node.js project:

```tsx
npm init -y
npm install --save-dev typescript
npm install @meshsdk/core
npx tsc --init
```

Update `tsconfig.json` with ES module settings:

```tsx
{
  "compilerOptions": {
    "target": "ESNext",
    "module": "ESNext",
    "moduleResolution": "Node",
    "outDir": "dist"
  }
}
```

Add the start script to `package.json`:

```tsx
{
  "type": "module",
  "scripts": {
    "start": "tsc && node ./dist/main.js"
  }
}
```

## How Do You Build the Minting Transaction?

Create `main.ts` and import the required packages:

```tsx
import {
  MeshWallet,
  MeshTxBuilder,
  ForgeScript,
  BlockfrostProvider,
  resolveScriptHash,
  stringToHex,
} from '@meshsdk/core';
import type { AssetMetadata } from '@meshsdk/core';


import { metadata } from './metadata.js';
import { recipients } from './recipients.js';
```

Configure your wallet and provider credentials (use your own keys for real deployments):

```tsx
const demoCLIKey = {
  paymentSkey:
    '5820aaca553a7b95b38b5d9b82a5daa7a27ac8e34f3cf27152a978f4576520dd6503',
  stakeSkey:
    '582097c458f19a3111c3b965220b1bef7d548fd75bc140a7f0a4f080e03cce604f0e',
};
const networkId = 0;
const blockfrostKey = 'BLOCKFROST_KEY_HERE';
```

Initialize the provider and wallet:

```tsx
const provider = new BlockfrostProvider(blockfrostKey);

const wallet = new MeshWallet({
  networkId: networkId,
  fetcher: provider,
  submitter: provider,
  key: {
    type: 'cli',
    payment: demoCLIKey.paymentSkey,
    stake: demoCLIKey.stakeSkey,
  },
});

const walletAddress = wallet.getPaymentAddress();
const forgingScript = ForgeScript.withOneSignature(walletAddress);
```

You can also use mnemonic phrases or private keys instead of CLI keys. See the <Link href="/apis/wallets/meshwallet">MeshWallet documentation</Link>.

Build the minting transaction using the Mesh TypeScript blockchain SDK by looping through recipients:

```tsx
const utxos = await wallet.getUtxos();
const policyId = resolveScriptHash(forgingScript);

const txBuilder = new MeshTxBuilder({
  fetcher: provider,
  verbose: true,
});

const txMetadata: { [policyId: string]: { [assetName: string]: AssetMetadata } } = {};
txMetadata[policyId] = {};

for (let recipient in recipients) {
  const recipientAddress = recipient;
  const assetName = recipients[recipient];
  const assetMetadata: AssetMetadata = metadata[assetName];
  const tokenNameHex = stringToHex(assetName);

  txMetadata[policyId][assetName] = assetMetadata;

  txBuilder
    .mint("1", policyId, tokenNameHex)
    .mintingScript(forgingScript)
    .txOut(recipientAddress, [{ unit: policyId + tokenNameHex, quantity: "1" }]);
}

txBuilder
  .metadataValue(721, txMetadata)
  .changeAddress(walletAddress)
  .selectUtxosFrom(utxos);

const unsignedTx = await txBuilder.complete();
const signedTx = await wallet.signTx(unsignedTx);
const txHash = await wallet.submitTx(signedTx);

console.log('Transaction submitted:', txHash);
```

Run the minting script:

```tsx
npm start
```

A successful execution returns a transaction hash. Your NFTs are now minted and distributed to the specified recipients.

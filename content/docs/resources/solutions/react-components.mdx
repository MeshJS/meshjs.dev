**TL;DR:** Mesh SDK provides production-ready React blockchain components for Cardano dApp development. This TypeScript blockchain SDK includes CardanoWallet for wallet connection, hooks like useWallet and useAssets for state management, and full type definitions. Install with `npm install @meshsdk/react` and add wallet connectivity in minutes.

---

Mesh React components are pre-built UI elements that solve the <Link href="/resources/challenges/wallet-integration">wallet integration challenges</Link> common in Cardano dApp development. This TypeScript blockchain SDK eliminates weeks of development by providing production-tested components for wallet connection, balance displays, and asset management. Instead of handling wallet variations, edge cases, and accessibility from scratch, you use components that work reliably across all major Cardano wallets. Whether you're building JavaScript Web3 applications or full-featured React blockchain interfaces, Mesh handles the complexity so you can focus on your dApp's unique features.

## How Do I Add Cardano React Components?

Add Cardano components to your React application in minutes:

**Step 1: Install the React package**

```bash
npm install @meshsdk/react
```

**Step 2: Add the provider**

```tsx
import { MeshProvider } from "@meshsdk/react";

function App() {
  return (
    <MeshProvider>
      <YourApp />
    </MeshProvider>
  );
}
```

**Step 3: Use components and hooks**

```tsx
import { CardanoWallet, useWallet, useLovelace } from "@meshsdk/react";

function Dashboard() {
  const { connected } = useWallet();
  const lovelace = useLovelace();

  return (
    <div>
      <CardanoWallet />
      {connected && <p>Balance: {Number(lovelace) / 1_000_000} ADA</p>}
    </div>
  );
}
```

## What React Components Does Mesh SDK Provide?

Mesh React provides a comprehensive component library for Cardano dApp development:

- **CardanoWallet**: Complete wallet connection UI with multi-wallet support
- **useWallet Hook**: Reactive wallet state and connection management
- **useAssets Hook**: Access connected wallet's native tokens and NFTs
- **useLovelace Hook**: Get wallet ADA balance with automatic updates
- **useAddress Hook**: Access connected wallet addresses
- **useNetwork Hook**: Detect connected network (mainnet/testnet)
- **MeshProvider**: Context provider for wallet state management
- **TypeScript Types**: Full type definitions for all components and hooks

## How Do I Use Each Component?

### CardanoWallet

CardanoWallet is the primary wallet connection component for Cardano dApp development:

```tsx
import { CardanoWallet } from "@meshsdk/react";

// Basic usage
<CardanoWallet />

// With customization
<CardanoWallet
  label="Connect Wallet"
  onConnected={() => console.log("Connected!")}
  isDark={true}
/>
```

CardanoWallet provides:
- Automatic wallet detection
- Wallet selection dropdown
- Connection state management
- Connected address display
- Disconnect functionality

### useWallet Hook

Access wallet state anywhere in your app:

```tsx
import { useWallet } from "@meshsdk/react";

function WalletInfo() {
  const {
    connected,      // boolean - true when wallet is connected
    connecting,     // boolean - true during connection attempt
    wallet,         // BrowserWallet instance for transactions
    name,           // string - connected wallet name (e.g., "Eternl")
    connect,        // function to connect by wallet ID
    disconnect,     // function to disconnect current wallet
  } = useWallet();

  return (
    <div>
      {connected ? (
        <p>Connected to {name}</p>
      ) : (
        <button onClick={() => connect("eternl")}>Connect</button>
      )}
    </div>
  );
}
```

### useLovelace Hook

Get the wallet's ADA balance:

```tsx
import { useLovelace } from "@meshsdk/react";

function Balance() {
  const lovelace = useLovelace();

  if (!lovelace) return null;

  const ada = Number(lovelace) / 1_000_000;
  return <p>{ada.toFixed(2)} ADA</p>;
}
```

### useAssets Hook

Access native tokens and NFTs:

```tsx
import { useAssets } from "@meshsdk/react";

function TokenList() {
  const assets = useAssets();

  if (!assets) return <p>Loading assets...</p>;

  return (
    <ul>
      {assets.map((asset) => (
        <li key={asset.unit}>
          {asset.assetName}: {asset.quantity}
        </li>
      ))}
    </ul>
  );
}
```

### useAddress Hook

Access wallet addresses:

```tsx
import { useAddress } from "@meshsdk/react";

function AddressDisplay() {
  const address = useAddress();

  if (!address) return null;

  return (
    <p title={address}>
      {address.slice(0, 8)}...{address.slice(-8)}
    </p>
  );
}
```

### useNetwork Hook

Detect the connected network:

```tsx
import { useNetwork } from "@meshsdk/react";

function NetworkBadge() {
  const network = useNetwork();

  return (
    <span className={network === 1 ? "mainnet" : "testnet"}>
      {network === 1 ? "Mainnet" : "Testnet"}
    </span>
  );
}
```

## How Do I Build Custom React Blockchain Components?

Mesh hooks enable fully custom UIs for your JavaScript Web3 applications while retaining all functionality:

```tsx
import { BrowserWallet } from "@meshsdk/core";
import { useWallet, useLovelace, useAssets } from "@meshsdk/react";

function CustomWalletPanel() {
  const { connected, wallet, connect, disconnect } = useWallet();
  const lovelace = useLovelace();
  const assets = useAssets();

  if (!connected) {
    const wallets = BrowserWallet.getInstalledWallets();

    return (
      <div className="wallet-selector">
        <h3>Select Wallet</h3>
        {wallets.map(w => (
          <button key={w.id} onClick={() => connect(w.id)}>
            <img src={w.icon} alt="" width={24} height={24} />
            {w.name}
          </button>
        ))}
      </div>
    );
  }

  return (
    <div className="wallet-panel">
      <div className="balance">
        <span>{(Number(lovelace) / 1_000_000).toFixed(2)}</span>
        <span>ADA</span>
      </div>
      <div className="assets">
        {assets?.length} tokens
      </div>
      <button onClick={disconnect}>Disconnect</button>
    </div>
  );
}
```

## What Frameworks Work with Mesh React Components?

Mesh React components work with all major React frameworks. Here's how to set them up:

### Next.js Setup

For Next.js applications, add MeshProvider to your client-side layout:

```tsx
// app/layout.tsx or _app.tsx
"use client";

import { MeshProvider } from "@meshsdk/react";

export default function RootLayout({ children }) {
  return (
    <html>
      <body>
        <MeshProvider>{children}</MeshProvider>
      </body>
    </html>
  );
}
```

### Vite/Create React App

Standard React setup works without additional configuration:

```tsx
// main.tsx
import { MeshProvider } from "@meshsdk/react";
import App from "./App";

ReactDOM.createRoot(document.getElementById("root")).render(
  <MeshProvider>
    <App />
  </MeshProvider>
);
```

Mesh is an open-source TypeScript blockchain SDK for building decentralized applications on Cardano. These guides walk you through wallet integration, NFT minting, smart contract development, and production deployment using JavaScript Web3 patterns familiar to React developers.

## What Can You Build with This Cardano SDK?

From basic wallet connections to complex smart contract interactions, these tutorials cover practical Cardano dApp development:

- **Getting Started**: Set up a Web3 application with Next.js and connect browser wallets
- **Minting & NFTs**: Create native tokens and mint NFT collections in Node.js or the browser
- **Smart Contracts**: Interact with Plutus and Aiken contracts for vesting, marketplaces, and custom logic
- **Multi-Signature Transactions**: Coordinate transactions requiring multiple wallet signatures
- **Production Deployment**: Security best practices and performance optimization for live dApps

Whether migrating from another Cardano SDK like Lucid or starting fresh, these tutorials help you ship React blockchain applications faster.

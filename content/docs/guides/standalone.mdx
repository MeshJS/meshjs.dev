Running standalone TypeScript scripts is the fastest way to interact with Cardano for testing, automation, and quick prototypes. Using `tsx`, you can execute TypeScript files directly without a build step—ideal for Cardano dApp development and blockchain scripting.

**What you'll build:**
- A minimal project setup with the Mesh Cardano SDK
- A script that creates a wallet and sends a transaction
- Zero-config TypeScript execution

## How Do You Set Up the Project?

Create `package.json` in your project root:

```tsx
{
  "type": "module",
  "dependencies": {},
  "scripts": {
    "dev": "tsx index.ts"
  }
}
```

Install the dependencies:

```tsx
npm install tsx @meshsdk/core
```

- `@meshsdk/core` — The Mesh Cardano SDK for TypeScript blockchain development
- `tsx` — Runs TypeScript directly without compilation

## How Do You Send a Transaction?

Create `index.ts` with the following code:

```tsx
import { BlockfrostProvider, MeshWallet, MeshTxBuilder } from "@meshsdk/core";

// Set up the blockchain provider with your key
const provider = new BlockfrostProvider("YOUR_KEY_HERE");

// Initialize the wallet with a mnemonic key
const wallet = new MeshWallet({
  networkId: 0,
  fetcher: provider,
  submitter: provider,
  key: {
    type: "mnemonic",
    words: [
      "your", "mnemonic", "...", "here",
    ],
  },
});

// Get UTxOs and change address
const utxos = await wallet.getUtxos();
const changeAddress = await wallet.getChangeAddress();

// Create and send a transaction
const txBuilder = new MeshTxBuilder({
  fetcher: provider,
});

const unsignedTx = await txBuilder
  .txOut("addr_test1qp2k7wnshzngpqw0xmy33hvexw4aeg60yr79x3yeeqt3s2uvldqg2n2p8y4kyjm8sqfyg0tpq9042atz0fr8c3grjmysdp6yv3", [{ unit: "lovelace", quantity: "1000000" }])
  .changeAddress(changeAddress)
  .selectUtxosFrom(utxos)
  .complete();

const signedTx = await wallet.signTx(unsignedTx);
const txHash = await wallet.submitTx(signedTx);
console.log('Transaction hash:', txHash);
```

## How Do You Run the Script?

Before running, replace the placeholder values:
- Get a Blockfrost API key from <Link href="https://blockfrost.io/">blockfrost.io</Link>
- Generate a mnemonic using <Link href="/apis/wallets/meshwallet#generate-wallet">MeshWallet.brew()</Link>

Execute the script:

```tsx
npm run dev
```

The transaction hash logs to the console upon success.
